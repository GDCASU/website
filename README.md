# VGDC Website
A website for the club. It is hosted from the HeatwaveAZ GitHub account at [heatwaveaz.github.io](heatwaveaz.github.io).

## Updating the Website
1. Push changes to the repository in the VGDC GitHub organization.
2. An officer must log into the HeatwaveAZ GitHub account.
3. Go to the forked version of the repository at [https://github.com/HeatwaveAZ/HeatwaveAZ.github.io](https://github.com/HeatwaveAZ/HeatwaveAZ.github.io).
4. Click "Fetch upstream" and then "Fetch and merge".
